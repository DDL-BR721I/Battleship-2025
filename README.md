Battleship 2025

A fully functioning Battleship game built with Python and Django, featuring interactive gameplay between the user and an AI opponent.
Features

- Object-Oriented Design: Classes for players, ships, shots, and boards ensure clean and modular code.  
- Game Logic Validation: Robust checks for ship placement, move validation, and hit/miss tracking.  
- AI Opponent: Implements multiple decision strategies (random, cross, line search) for adaptive gameplay.  
- Dynamic Frontend: Interactive UI using Django templates and JavaScript, showing real-time board updates.  
- Software Engineering Practices: Modular design, reusable components, and input validation for maintainability and scalability.
